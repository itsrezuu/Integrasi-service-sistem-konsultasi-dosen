## Run Locally

Clone the project

```bash
  git clone https://github.com/itsrezuu/Integrasi-service-sistem-konsultasi-dosen.git
```

Go to the project directory

```bash
  cd Integrasi-service-sistem-konsultasi-dosen
  cd konsultasi-service
```

Install dependencies

```bash
  npm install
```

Environment

```bash
  cp .env.example .env  
```

Database

```bash
  create new database: konsultasi_service
  import sql file to your database client 
```

Start the server

```bash
  npm start
```
